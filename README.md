# anima

First fully on-chain sovereign agent runtime on 0G.

## What it is

Anima is a CLI-hosted agent runtime where the agent's identity, memory, reasoning, and economic life all live on 0G's decentralized infrastructure. Operator runs `anima init` once. After that, the agent persists on chain: close the laptop, walk away, the agent survives. Any operator machine can re-attach via the iNFT.

## Architecture

Six-layer stack, each layer anchored to a 0G primitive:

| Layer | Lives on | How |
|-------|----------|-----|
| Identity | 0G Chain | ERC-7857 iNFT |
| Memory | 0G Storage | KV for mutable state, blob for immutable |
| Brain | 0G Compute (TeeML) | OpenAI-compat inference, any live model |
| Harness | 0G Sandbox (TDX) | Attested orchestration layer |
| Limbs | User machines | Filesystem + shell access via paired daemon |
| Wallet | Hybrid | Runtime hot copy + iNFT-metadata cold copy |

## `anima init`

Runs the first-time onboarding wizard. Two wallets are involved per project-anima.md section 22.1:

- **Operator wallet** — owns the iNFT. Dev pattern: macOS keychain under the service name `dev.deployer`. Production will support MetaMask / WalletConnect / hardware wallet / keystore file (pluggable via the `OperatorSigner` interface).
- **Agent EOA** — separate fresh key generated by the wizard. Pays infra gas (subname claim, memory sync, future tool txs).

Flow:
1. Pick network.
2. Pick a subname (optional).
3. Set passphrase for the agent keystore.
4. Wizard generates a fresh agent EOA and stores the encrypted keystore locally.
5. Operator wallet (auto-loaded from keychain) mints an ERC-7857 iNFT to itself with 6 canonical `IntelligentData` slots (`memory-index`, `identity`, `persona`, `profile`, `keystore`, `activity-log`) AND calls `setApprovalForAll(agent, true)` so the agent can push memory updates later without the operator's key.
6. Operator funds the agent EOA with 0.03 0G for ongoing infra.
7. If a subname was chosen, agent claims `<subname>.anima.0g` via the permissionless registrar, sets `address` + `agent:inft` text records.
8. Writes `anima.config.ts` with `identity.iNFT`, `identity.operator`, `identity.agent` populated.

If the operator wallet has zero gas, the wizard prints a "fund me, re-run" note and skips mint + subname gracefully.
